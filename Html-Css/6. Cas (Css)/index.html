Krecemo sa Stilovima (Css-om)

Css (Cascading Style Sheets) je jezik koji koristimo za styilizoanje HTML dokumenta.

Css mozemo primeniti na 3 nacina:

  1. Inline Css
  2. Stavljanje style taga unutar head
  3. Eksterni Css
